# Penetration-Testing-Report-Metasploitable-Lab
Project Overview  This project demonstrates a full penetration testing engagement conducted in a controlled lab environment. The objective was to identify vulnerabilities in a target system, exploit them, and demonstrate post-exploitation activities such as gaining system-level access and creating a new user account.



⚠️ Disclaimer

This assessment was performed strictly in a controlled and authorized environment for educational purposes only. Unauthorized testing on systems without explicit permission is illegal and unethical.
